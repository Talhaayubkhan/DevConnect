# DevConnect 💻❤️

Welcome to DevConnect! 🎉 A Tinder-inspired platform built just for developers! DevConnect connects coders, programmers, and tech lovers to meet, chat, and team up. Whether you’re hunting for a coding soulmate 👩‍💻👨‍💻, a project collaborator, or just want to nerd out over code, this is your spot! ✨

## Features 🌟

- **Profile Matching** 👥: Swipe through dev profiles based on skills, interests, and vibes!
- **Real-Time Chat** 💬: Instant messaging with your matches.
- **Skill Filters** 🔍: Find devs by tech stack (e.g., JS, Python, etc.).
- **Collab Mode** 🤓: Share ideas and build projects together!
- **Dev Community** 🌐: Network and grow in a coder-only space.

## Installation ⚙️

### Prerequisites ✅

- Node.js (v16.x+)
- MongoDB (local or Atlas ☁️)
- npm or yarn 🧶

### Steps 🏃‍♂️

1. **Clone the Repo**

   ```bash
   git clone https://github.com/your-username/devconnect.git
   cd devconnect
   ```

2. **Backend Setup** 🖥️

   - Jump to the backend:
     ```bash
     cd backend
     ```
   - Install deps:
     ```bash
     npm install
     ```
   - Add a `.env` file:
     ```
     PORT=5000
     MONGODB_URI=your_mongodb_uri
     JWT_SECRET=your_secret_key
     ```
   - Fire it up:
     ```bash
     npm start
     ```

3. **Frontend Setup** 🎨
   - Head to the frontend:
     ```bash
     cd frontend
     ```
   - Install deps:
     ```bash
     npm install
     ```
   - Launch it:
     ```bash
     npm start
     ```
